# **二叉树是最重要的基本数据结构，没有之一。**
一叉落万叉生:
    1. 红黑树
    2. 多叉树
    3. 图 -> 邻接表
    4. 字典树
    5. 并查集
    6. 线段树

- **完全二叉树**: 一种特殊的二叉树，其中每一层除了最后一层外，所有节点都被完全填满，并且最后一层的所有节点都尽可能地向左对齐。(e.g: 底层为 第h层, 则第h-1层的节点数为$2^(h-1)$, 第h层的节点数在$1~2^(h-1)$之间)
- **二叉搜索树**(BST): 一种有序的二叉树
  - 满足:
    1. 左子树上所有节点的值均小于根节点的值
    2. 右子树上所有节点的值均大于根节点的值
    3. 左右子树也分别为二叉搜索树
- **平衡二叉树(AVL树)**: 一种特殊的二叉搜索
  - 满足: 任意节点的左右子树的高度差不超过1
- **自平衡二叉树**: 通过在插入和删除节点时自动调整树的结构来保持平衡的二叉搜索树(e.g: 红黑树, AVL树) 主要通过旋转实现
